Willkommen zu Tauri!

Entdecke, wie Tauri eine sichere, effiziente Alternative zu Electron bietet.

  • Rust-basiert: Maximale Sicherheit durch kompilierte System-APIs
  • Kleine Bundle-Größe: Nur 3 - 5 MB statt >100 MB
  • Native Performance: Ohne den Chromium-Overhead von Electron
  • Cross-Plattform: Windows, macOS & Linux mit ein und demselben Code
  • Web-Technologien: HTML/CSS/JS bleiben vertraut, während das Backend in Rust läuft
