Wie funktioniert Tauri?

API & Fähigkeiten

  • Zugriff auf das Rust-Backend über die Tauri-APIs
  • Fenster-Manipulation (Größe, Position, Always-on-Top, …)
  • Native Dialoge nutzen (Datei-Öffnen, Speichern, Alerts)
  • Weitere System-APIs: Notifications, Clipboard, Menüs, u.v.m.

Projektstruktur

  • src/ – Frontend (HTML, CSS, JS/TS & Framework)
  • src-tauri/ – Rust-Backend und Tauri-Konfiguration
  • Grundlegend ein einfaches JavaScript-Projekt mit Tauri-CLI-Integration
  • Build-Prozess: Frontend → dist/ → Rust-Binary bündelt alles
